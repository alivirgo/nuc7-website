Module 01

Physical Layer

Welcome to the foundation of the OSI model. The Physical Layer is responsible for the actual electrical, optical, or radio-frequency transmission of raw bits across a physical medium.

Core Concepts

Think of this layer as the "pipes" of the internet. It deals with bit-by-bit delivery and defines the hardware specifications like cables, cards, and connectors.

Module 02

Data Link Layer

This layer ensures reliable delivery of data across a single link between two devices. It introduces the concept of **Frames** and **MAC Addresses**.

Switching and Frames

While the physical layer sees bits, the data link layer sees structured frames with error-checking mechanisms.

Module 03

Network Layer

The Network Layer handles logical addressing (**IP Addresses**) and global routing of packets.

Transport Layer
